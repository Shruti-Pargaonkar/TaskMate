# TaskMate

DESCRIPTION:
TaskMate is a task management application designed to help users create, track, and manage daily tasks efficiently.

FEATURES:
1. Create, update, and delete tasks
2. Task status management (pending / completed)
3. User-specific task handling
4. RESTful APIs for task operations

TECH STACK
Backend: Java, Spring Boot, Spring Data JPA
Frontend: React.js
Database: MySQL
Tools: Postman

APPLICATION FLOW:
User performs task actions from UI
Requests are processed through REST APIs
Tasks are stored and fetched from database
Responses returned as JSON
